# 🚗 Uber Stock Analysis

## 📌 Overview

This project analyzes Uber’s historical stock performance using Python, with a focus on transforming raw market data into actionable business insights.

Through time-series analysis, statistical metrics, and data visualization, this project demonstrates how financial data can be used to understand trends, volatility, and overall company performance.

---

## 🎯 Objective

The goal of this project is to:

* Analyze Uber stock price trends over time
* Identify patterns in trading volume and volatility
* Generate meaningful insights for business and investment decision-making
* Present findings in a clean, professional, and portfolio-ready format

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib / Seaborn
* Jupyter Notebook

## 📊 Key Analysis & Visualizations

The project includes the following visualizations:

* 📈 Stock Price Trend Over Time
* 📊 Trading Volume Analysis
* 📉 Moving Averages (7-day & 30-day)
* 📊 Daily Returns Distribution
* 📉 Volatility Trends

Each visualization is designed to highlight important aspects of Uber’s stock performance and support data-driven conclusions.

---

## 💡 Key Insights

* Uber’s stock shows periods of rapid growth followed by volatility, indicating sensitivity to market conditions
* Trading volume spikes often align with major price movements
* Rolling averages help smooth short-term fluctuations and reveal longer-term trends
* Volatility analysis highlights risk patterns that are important for investors and stakeholders

## ▶️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/your-username/uber-stock-analysis.git
```

2. Navigate into the project folder:

```bash
cd uber-stock-analysis
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Open the notebook:

```bash
jupyter notebook
```
